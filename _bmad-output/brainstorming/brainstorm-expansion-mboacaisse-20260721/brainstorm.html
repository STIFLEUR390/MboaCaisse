Synthèse

Le Fil Rouge & les Priorités

Ce qui reste sur la table après 57 lignes de memlog.

📱 Le téléphone = clé universelle

Wallet + fidélité + reconnaissance + historique. Pas de login, pas de carte, pas d'app. Juste un numéro. Le client arrive, son téléphone ouvre tout.

Top 3 Priorités

  1. Wallet noyau dur + ledger append-only

    Le wallet est la source de vérité. Ledger INSERT-only, backup 5 min. Tout le reste est cosmétique sans ça.

  2. Impression queue async

    Maillon critique de crédibilité pro. Retry + fallback ticket numérique. Sans impression fiable, pas de passage en production.

  3. Feature flags via licensing Ed25519

    6 bundles = même code, flags dès maintenant. Le licensing P4 existe déjà. Codé tôt = pas de refactor.

Synergie : le Triangle Résilient

📒
Ledger

INSERT-only, backup 5 min. Trace immuable de chaque transaction.

💼
Wallet

Solde local, multi-sources. Source de vérité pour le paiement.

🖨
Impression Queue

Async, retry, fallback numérique. Crédibilité professionnelle.

Chacun tient si un autre tombe. Le wallet enregistre dans le ledger et déclenche l'impression. Le ledger peut être rejoué. L'impression peut tomber en ticket numérique. Le wallet tourne en local.

Traiteur QR takeaway, Resto QR table. Même logique, vue différente. Un seul menu public sert tous les bundles.
